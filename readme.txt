INTRODUCTION
------------

Cette application web permet de répondre à la demande d'établissements scolaires
qui suite  à  la  réforme  des  rythmes  scolaires  les  établissements  scolaires
en   collaboration   avec   les   communes   ont   décidé   de   se   doter   d’une
application  web  permettant  d’automatiser  la  gestion  des  temps  d’activités
périscolaires (TAP) : garderie, cantine et activités d’animation.

INSTALLATION
------------

 * Faire tourner le script install_bd.sql pour créer les bases de données
 nécessaires au bon fonctionnement du site web

 * Ouvrir le projet avec netbeans

CONFIGURATION
-------------

 * Configurer un serveur tomcat

 * Ajouter le serveur tomcat à l'application netbeans

 * Ajouter la connexion Oracle à l'application netbeans

 * Compiler et deployer en utilisant netbeans
